# ComplianceCI v2 - Veille Réglementaire Premium

## 🚀 Déployer sur Vercel en 2 minutes

### Étape 1 : Créer un repo GitHub

1. Allez sur **github.com** → **New Repository**
2. Nom : `complianceci-v2`
3. Public
4. Create

### Étape 2 : Upload les fichiers

1. **Add file** → **Upload files**
2. Drag-drop TOUS les fichiers de ce dossier
3. **Commit**

### Étape 3 : Déployer sur Vercel

1. **vercel.com** → **New Project**
2. Sélectionnez repo `complianceci-v2`
3. **Import** → **Deploy**

**SITE LIVE en 2-3 minutes ! ✨**
